### 2. 首页提醒引擎重构 (Home Reminders)
- [x] **纪念日提醒**：判断今天是否为猫咪的生日或到家纪念日，如果是，则在提醒面板中置顶显示特殊庆祝卡片（如“今天是岁岁3岁生日！🎂”）。

    - 如果超期，显示**“已逾期 X 天 ⚠️”**。

### 3. 图标异常排查与修复 (Meow IP)
- [x] **问题定位**：目前的 `meow-ip.png` 体积过大（达到 10MB），导致 Web 端和 PWA 加载异常或无法显示。
- [x] **修复方案**：对图标进行压缩和尺寸裁剪（建议 512x512，小于 200KB），以符合 PWA Manifest 和网页优化的标准。

## 5. V5 全局 UI/UX 严格校准计划 (Source: TASK_V5.md)

### Task 1: 消除所有违规的视觉质感 (扁平化回归)
- [x] **移除渐变色**：全局搜索并清理所有的 `linear-gradient`，包括但不限于 `.btn-drawer-save` 按钮、纪念日提醒卡片背景、里程碑图标背景。全部替换为纯色块（品牌主色或辅助色）。
- [x] **统一阴影规范**：检查并重置所有 `box-shadow`。移除花哨的彩色投影（如蓝色、黄色投影），强制所有元素的投影遵循 Spec 规定的极轻微软投影：`0 2px 8px rgba(0, 0, 0, 0.06)`（即变量 `--shadow-base` 或 `--shadow-soft`）。

### Task 2: 严格强制执行 8pt 网格与卡片间距
- [x] **卡片内边距校准**：修改 `main.css` 和 `components.css` 中的 `.card` 样式，将其 `padding` 从 `24px` 统一收紧至 Spec 规定的 **16px** (`--spacing-16`)。
- [x] **卡片内部元素间距**：确保卡片内部元素（如标题与内容）的 `gap` 或 `margin-bottom` 严格使用 **12px** (`--spacing-12`)。全局排查清理内联样式中的非标间距。

### Task 3: 圆角规范的收敛 (12-16pt 标准)
- [x] **修复夸张圆角**：将顶部 Banner 的 `24px` 底部圆角、底部 Tabbar 的 `20px` 顶部圆角，以及抽屉弹窗的 `28px` 圆角，全部降级统一至最大 **16px** (`--radius-16`) 以符合卡片组件规范。
- [x] **消除硬编码圆角**：排查 `ui.js` 中所有内联编写的 `border-radius: 14px`、`12px` 等硬编码，将其全部绑定到 CSS 变量（`var(--radius-12)` 或 `var(--radius-16)`）上。

### Task 4: 字体排版与组件细节打磨
- [ ] **大标题与对齐复查**：确保“系统设置”、“我的”、“记录”页面的主标题视觉层级和对齐方式严格符合规范。
- [x] **UI 渲染清理**：检查 `ui.js` 内部生成 HTML 字符串的结构，剔除冗余嵌套，用纯净的 CSS 类名驱动样式，而不是堆砌繁杂的行内 style。
