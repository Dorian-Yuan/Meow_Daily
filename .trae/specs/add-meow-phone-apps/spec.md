# Meow Phone 新增轻量小应用调研 Spec

## Why
Meow Phone 目前仅有「猫抓老鼠」和「像素画板」两个小应用，类型覆盖有限（策略类 + 创意类）。作为隐藏彩蛋系统，增加更多轻量、猫主题的小应用可以提升探索惊喜感和可玩性，让用户每次打开都有新发现。

## 现有应用分析

| 应用 | 类型 | 交互模式 | 代码量 |
|------|------|----------|--------|
| 🐭 猫抓老鼠 | 策略/逻辑 | 单击插旗 + 双击翻开 | ~300行 |
| 🎨 像素画板 | 创意/工具 | 拖拽绘制 + 调色 | ~500行 |
| ⚙️ 设置 | 系统/配置 | 表单选择 | 内嵌主文件 |

**现有缺口**：缺少反应类、休闲解压类、趣味互动类、实用工具类应用。

## 调研推荐：6个新增小应用

### 1. 🧶 毛线球（Yarn Ball）— 休闲解压类

**核心玩法**：屏幕上有一个彩色毛线球，用户可以拖拽/甩动它，毛线球会在屏幕内弹跳，留下渐隐的彩色轨迹。支持切换毛线球颜色，轨迹颜色随之变化。

**推荐理由**：
- 极致轻量，核心仅依赖 CSS transform + requestAnimationFrame
- 猫与毛线球是天然关联，主题契合度最高
- 纯解压交互，与现有应用形成差异化（无需思考、无需策略）
- 视觉效果治愈，符合产品"年轻化治愈系"调性

**交互设计**：
- 触摸/鼠标拖拽毛线球
- 松手后根据甩动速度继续惯性运动
- 碰到屏幕边缘反弹
- 轨迹渐隐效果（保留最近 30 个位置点）
- 底部工具栏：切换颜色 🎨、清除轨迹 🗑️

**预估代码量**：~150行

---

### 2. 🎲 猫咪占卜（Cat Fortune）— 趣味互动类

**核心玩法**：用户点击一只闭眼的猫咪，猫咪"思考"片刻后睁开眼，给出一条今日运势（随机抽取），配合不同的猫咪表情和运势等级（大吉/中吉/小吉/末吉/凶）。

**推荐理由**：
- 代码极简，核心是随机文案 + CSS 动画
- 每日可玩一次（基于日期缓存结果），增加回访动力
- 猫咪视角的运势文案极具情绪价值，完美契合产品调性
- 可以与主应用的"提醒"功能产生联动想象

**交互设计**：
- 初始：闭眼猫咪 + "点我占卜" 提示
- 点击后：猫咪思考动画（2秒），眼睛转动
- 结果：猫咪睁眼 + 运势等级 + 趣味文案
- 每日运势缓存到 localStorage，同一天结果不变
- 底部"再摇一次"按钮（仅改变文案，运势等级不变）

**运势文案示例**：
- 大吉 🌟："今天你是全宇宙最幸运的猫！铲屎官会给你加罐头！"
- 中吉 ✨："运势不错喵~适合在阳光下打个盹"
- 末吉 🍀："平平淡淡才是真，今天适合安静地舔毛"
- 凶 😾："小心！可能会被强制洗澡...快躲起来！"

**预估代码量**：~120行

---

### 3. 🧩 猫咪翻牌（Cat Memory）— 记忆益智类

**核心玩法**：经典翻牌配对游戏，使用猫主题图案（🐱😺😸😹😻😼😽🙀😿😾 + 🐟🧶🐾🐭），4×4 共 8 对卡片，翻开两张相同即消除。

**推荐理由**：
- 经典游戏机制，用户零学习成本
- 猫咪表情符号天然适合做卡片图案，视觉丰富
- 填补"记忆/益智"类空白，与扫雷的策略类不同
- 游戏时长适中（1-3分钟），适合碎片时间

**交互设计**：
- 4×4 卡片网格（8对）
- 点击翻开，两张匹配则消除，不匹配则翻回
- 计步数 + 计时
- 全部消除后显示成绩 + 星级评价
- 支持 3×4（6对）简单模式

**预估代码量**：~200行

---

### 4. 🎯 打地鼠（Whack-a-Mouse）— 反应动作类

**核心玩法**：3×3 的地鼠洞，老鼠随机从洞中探出，用户在限定时间内点击老鼠得分。打中显示猫咪爪印，漏掉则老鼠偷笑逃走。

**推荐理由**：
- 猫抓老鼠主题的完美延伸，与"猫抓老鼠"扫雷形成系列
- 反应类游戏填补动作类空白
- 节奏紧张有趣，30秒一局，碎片化友好
- 视觉反馈强烈（爪印 + 得分飘字），爽感十足

**交互设计**：
- 3×3 地鼠洞网格
- 30秒倒计时
- 老鼠出现 → 停留 0.8-1.5秒 → 缩回
- 打中：+10分 + 爪印特效 + "喵！"飘字
- 漏掉：老鼠偷笑动画
- 结束：显示得分 + 评级（猫爪等级 🐾🐾🐾）
- 难度递进：后期老鼠停留时间缩短

**预估代码量**：~200行

---

### 5. 🎵 喵喵琴（Meow Piano）— 音乐创意类

**核心玩法**：一排猫咪按键（8个音阶），点击/滑动发出不同音调的"喵"声，可以弹奏简单旋律。支持录制和回放。

**推荐理由**：
- 与像素画板形成"视听双通道"创意工具组合
- 猫叫声音效极具趣味性和传播性
- Web Audio API 生成音调，无需外部音频文件
- 录制回放功能增加可玩深度

**交互设计**：
- 8个猫咪按键（Do Re Mi Fa Sol La Si 高Do）
- 每个按键不同颜色的猫咪头像
- 点击发出对应音调的合成"喵"声
- 支持滑动弹奏（手指滑过多个键）
- 录制按钮：记录弹奏序列
- 回放按钮：自动重放录制的旋律
- 音色切换：喵声 / 纯音 / 钢琴

**预估代码量**：~180行

---

### 6. ⏰ 喵喵计时器（Meow Timer）— 实用工具类

**核心玩法**：简约倒计时器 + 秒表，带有猫咪动画。倒计时结束时猫咪跳起来"喵！"提醒。

**推荐理由**：
- 唯一纯实用型应用，提升 Meow Phone 的"工具"属性
- 与主应用的"提醒"功能概念呼应
- 极简实现，核心是 setInterval + CSS 动画
- 猫咪动画让枯燥的计时变得有趣

**交互设计**：
- 两个模式切换：倒计时 ⏰ / 秒表 ⏱️
- 倒计时：预设选项（1/3/5/10/25分钟）+ 自定义
- 秒表：开始/暂停/重置 + 计次
- 倒计时进行中：猫咪打盹动画，剩余时间越少猫咪越不安
- 倒计时结束：猫咪跳起 + 震动反馈（如果支持）
- 圆形进度环显示剩余时间

**预估代码量**：~160行

## 推荐优先级排序

| 优先级 | 应用 | 理由 |
|--------|------|------|
| P0 | 🧶 毛线球 | 主题契合度最高，实现最简，纯解压差异化最大 |
| P0 | 🎲 猫咪占卜 | 代码量最少，情绪价值最高，每日回访动力 |
| P1 | 🎯 打地鼠 | 与"猫抓老鼠"形成系列，动作类填补空白 |
| P1 | 🧩 猫咪翻牌 | 经牌益智类，零学习成本 |
| P2 | 🎵 喵喵琴 | 创意类与像素画板互补，但音效实现稍复杂 |
| P2 | ⏰ 喵喵计时器 | 实用但趣味性相对最低 |

## What Changes
- 在 `js/modules/games/` 下新增各小应用的独立模块文件
- 在 `meow_phone.js` 的 `APP_REGISTRY` 中注册新应用
- 在 `css/components.css` 中添加各应用的样式
- 在 `store.js` 中扩展 `game_prefs` 以支持新应用的设置持久化

## Impact
- Affected code: `meow_phone.js`, `css/components.css`, `store.js`, 新增游戏模块文件
- APP_REGISTRY 图标网格当前仅 2 个应用，新增后布局更充实（4×2 = 8格，6个应用+设置刚好填满两行）

## ADDED Requirements

### Requirement: 毛线球应用
系统 SHALL 在 Meow Phone 中提供「毛线球」小应用，用户可拖拽毛线球在屏幕内弹跳并留下渐隐轨迹。

#### Scenario: 拖拽毛线球
- **WHEN** 用户触摸/点击毛线球并拖动
- **THEN** 毛线球跟随手指移动，松手后按惯性继续运动并在边缘反弹

#### Scenario: 切换颜色
- **WHEN** 用户点击颜色切换按钮
- **THEN** 毛线球和轨迹颜色切换为下一个预设颜色

### Requirement: 猫咪占卜应用
系统 SHALL 在 Meow Phone 中提供「猫咪占卜」小应用，用户点击猫咪获取每日运势。

#### Scenario: 每日占卜
- **WHEN** 用户首次当天点击占卜猫咪
- **THEN** 显示思考动画后展示随机运势结果，结果缓存至当天

#### Scenario: 重复占卜
- **WHEN** 用户当天再次进入占卜应用
- **THEN** 显示当天已缓存的运势结果，不重新随机

### Requirement: 猫咪翻牌应用
系统 SHALL 在 Meow Phone 中提供「猫咪翻牌」记忆配对游戏。

#### Scenario: 配对成功
- **WHEN** 用户翻开两张相同图案的卡片
- **THEN** 两张卡片消除，步数+1

#### Scenario: 游戏完成
- **WHEN** 所有卡片配对消除
- **THEN** 显示总步数、用时和星级评价

### Requirement: 打地鼠应用
系统 SHALL 在 Meow Phone 中提供「打地鼠」反应类小游戏。

#### Scenario: 击中地鼠
- **WHEN** 用户在老鼠探出期间点击老鼠
- **THEN** 显示爪印特效，得分+10

#### Scenario: 游戏结束
- **WHEN** 30秒倒计时结束
- **THEN** 显示最终得分和猫爪等级评价

### Requirement: 喵喵琴应用
系统 SHALL 在 Meow Phone 中提供「喵喵琴」音乐创意小应用。

#### Scenario: 弹奏音符
- **WHEN** 用户点击琴键
- **THEN** 发出对应音调的合成喵声

#### Scenario: 录制回放
- **WHEN** 用户点击录制后弹奏，再点击回放
- **THEN** 系统自动按录制时序重放弹奏序列

### Requirement: 喵喵计时器应用
系统 SHALL 在 Meow Phone 中提供「喵喵计时器」实用工具。

#### Scenario: 倒计时结束
- **WHEN** 倒计时归零
- **THEN** 猫咪跳起动画 + 视觉提醒

#### Scenario: 秒表计次
- **WHEN** 用户在秒表运行中点击计次
- **THEN** 记录当前时间点并继续计时
